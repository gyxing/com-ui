# simui

* 基于Vue2的一套UI组件

* [API文档入口](https://gyxing.github.io/simui/docs)

```base
npm install --save simui
```
```base
import Vue from 'vue'
import simui from 'simui'
import 'simui/dist/simui.css'

Vue.use(simui)
```

```base
<template>
    <div>
        <Icon type="reload" color="#f00" size="24" />
        <Button>click me</Button>
    </div>
</template>
```
